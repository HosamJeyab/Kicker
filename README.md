# ⚽ Kicker

**Kicker** is a sleek, fun, and data-driven football prediction application built with Flutter. It is designed for football fans who love to simulate matches and predict win/loss probabilities between international and Arab national teams using a refined algorithm inspired by official FIFA World Rankings and the classic ELO rating system.

No more endless arguments with your friends about who has the stronger squad—let **Kicker** settle it with numbers! 🚀

---

## ✨ Features

* 🔮 **Smart & Realistic Predictions:** Calculates logical win/loss/draw percentages based on real-time team strength data, completely avoiding unrealistic or exaggerated gaps.
* 🔍 **Instant Search & Filter:** Features an advanced, native search-and-select drop-down experience (`SearchAnchor`) that handles hundreds of national teams seamlessly with blistering performance.
* 🎨 **Modern Sports UI:** A gorgeous, eye-friendly Dark Theme designed to capture the excitement and energy of modern football apps.
* 📦 **Zero-Dependency Core:** Built using standard, native Material 3 widgets, ensuring high maintainability, top-tier performance, and clean code architecture.

---

## 🛠️ Tech Stack

* **Framework:** Flutter (Latest Version)
* **Language:** Dart
* **Architecture:** Structured and Clean Code Architecture
* **Algorithm:** Modified ELO Rating System (FIFA World Ranking Logic)
